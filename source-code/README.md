# Source Code Structure

## frontend/
Contains frontend application source code.

Examples:
- React
- Flutter
- Angular
- Vue

---

## backend/
Contains backend application source code.

Examples:
- Node.js
- Laravel
- Django
- Spring Boot

---

## database/
Contains database scripts and schema files.

Examples:
- SQL scripts
- ER diagrams
- migrations

---

## docs/
Technical documentation.

Examples:
- API documentation
- architecture
- deployment guides

---

## assets/
Static assets.

Examples:
- images
- icons
- videos

---

## tests/
Testing files.

Examples:
- unit tests
- integration tests
- UI tests

---

## scripts/
Automation and deployment scripts.

Examples:
- backup scripts
- deployment scripts
- setup scripts
